\doxysubsection{Detailed Description}
Dislocation trajectory analysis module. 

This module contains functions that are used in the script "{}dislocation\+\_\+script.\+py"{}.

The functions contained in this module allow the user to analyse Molecular Dynamics (MD) trajectories of straight edge dislocations generated using LAMMPS.

The script "{}ovito\+\_\+disloc.\+py"{} post-\/processes dislocation trajectories generated as LAMMPS dumps files containing the unwrapped coordinates of atoms. The functions in this script use as input the text files generated by "{}ovito\+\_\+disloc.\+py"{}, the format of which is described in further detail in its documentation.

The function "{}track\+\_\+disloc"{} included here allows the user to track dislocations across the various frames of a trajectory, since in a simulation containing multiple dislocations, different dislocations are indexed differently by the DXA tool. Accordingly, this function allows the user to "{}track"{} a given dislocation throught a trajectory and analyse its dynamics.

The script "{}dislocation\+\_\+script.\+py"{} and its documentation show how the other functions in this module can be used to carry out the necessary processing of dislocation trajectories prior to carrying out the dislocation tracking using "{}track\+\_\+disloc"{}.

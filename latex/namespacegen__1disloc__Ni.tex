\doxysubsection{Detailed Description}
Script to generate a single a/2\texorpdfstring{$<$}{<}110\texorpdfstring{$>$}{>}\{111\} edge dislocation in a pure FCC Ni cell. 

To construct the dislocation, two slabs of pure Ni are created, where one of the slabs has an extra plane (half-\/plane) of atoms. On relaxing this structure (see \doxylink{lmp__pre__strain_8py}{lmp\+\_\+pre\+\_\+strain.\+py}) the correct dislocation core structure, where the dislocation dissociates into two a/6\texorpdfstring{$<$}{<}112\texorpdfstring{$>$}{>} partial dislocations should form.

It is recommended that the cell dimensions here are kept unchanged and creating a bigger cell can be achieved by adding padding using \doxylink{gen__111__atoms_8py}{gen\+\_\+111\+\_\+atoms.\+py}.

The output structure is written to a .data file, which can then be read be used as an input to LAMMPS.

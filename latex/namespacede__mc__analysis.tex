\doxysubsection{Detailed Description}
Script to analyse the ouput from the parameter fitting using DE-\/\+MC. 

This script uses the functions contained in the \doxylink{funcs_8py}{funcs.\+py} module to\+:
\begin{DoxyItemize}
\item Calculate relevant parameter statistics using accepted samples obtained from fitting using DE-\/\+MC.
\item Visualise the DE-\/\+MC output as well as the marginal and joint parameter posterior distributions.
\item Produce a trace plot and parameter mean and convergence with DE-\/\+MC generation (iteration) to guage the convergence of the DE-\/\+MC.
\end{DoxyItemize}

The function, stat\+\_\+convergence, takes a long time to run. Accordingly, it is recommended that it is run only once, and its output can be written to a text file by setting write\+\_\+out=True in the function argument for further analysis.
